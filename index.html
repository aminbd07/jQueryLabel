jQLabel

Create labels like GMail with this awesome jQuery plugin, easily

What is jQLabel?

Using jQLabel you can create labels which are similar to GMail labels in look-n-feel. You can customize the background color of these labels based on their content or by their class name . jQLabel is released under New BSD license.

Using jqlabel is easy. Just set class name to your dom elements like <span/>, <p/>, <a/> like the following block.

Example HTML

Create <span class="label">labels</span> like <span class="label">GMail</span>
with this <span class="label">awesome</span> jQuery plugin, easily

Now call jqlabel plugin in your $(document).ready() method like this

Example Javascript

$(document).ready(function(){
    $(".label").jQLabel({"pointer":true,
                         "labels":{
                            "awesome":"#FF9E00",
                            "gmail":"#A65D03",
                            "labels":"#A49774"
                         }
    });
})

Output will be like - Create labels like GMail with this awesome jQuery plugin, easily


Using by class name

Instead of creating labels by using content, you can simply convert them by element's class name. For example check out the following HTML and Javascript block.

Example HTML

An <span class="important">Important</span> Label has
some  <span class="important">important content</span> in it

Now call jqlabel plugin in your $(document).ready() method like this

Example Javascript

$(document).ready(function(){
    $(".important").jQLabel();
})

And output will look like - An Important Label has some important content in it


Available Options

color   Specify the text color of the label using color option. By default the value of this options is set to "#FEFEFE" 

$(".important").jQLabel({color:"#DDD"});

backgroundColor   Specify the background color of the label using backgroundColor option. By default it's value is set to "#CE0000" 

$(".important").jQLabel({backgroundColor:"#356781"});

pointer   if set to true, the cursor's style will be "pointer" when someone hovers mouse over the labels. 

$(".important").jQLabel({pointer:false});
